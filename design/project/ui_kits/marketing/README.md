# TimeMatch Marketing UI Kit

High-fidelity recreation of the **time-match.tech** landing page — built against the TimeMatch Design System.

## What's here

- `index.html` — the demo. Loads all components and renders the full landing page.
- `Header.jsx` — sticky top nav with blur-on-scroll
- `Hero.jsx` + `ScheduleGrid.jsx` — display headline + the signature animated timetable
- `LogoBar.jsx` — partner / customer university logo strip
- `HowItWorks.jsx` — three-step explainer
- `Features.jsx` — feature grid with hairline cards
- `Stats.jsx` — big tabular-figure numerical claims
- `Testimonial.jsx` — single editorial pull quote
- `CTA.jsx` — dark inverted section with the demo request
- `Footer.jsx` — sitemap footer
- `atoms.jsx` — `<Button>`, `<Eyebrow>`, `<Section>`, `<Container>` building blocks

## How it reads

Pure marketing site: a fictional, in-system rendering of what time-match.tech would look like. No real customer logos, no real screenshots — university names and quotes are placeholders that match the tone documented in the brand README.
